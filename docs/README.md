# 📚 Documentation

This folder contains supporting documentation for the Formula 1 Driver Performance Intelligence project.

## Files

### 📄 SQL_Query_Explanations.md
Provides an overview of each SQL script, including its purpose, key SQL concepts used, and the business objective behind the queries.

### 📄 Data_Dictionary.md *(Coming Soon)*
Describes each dataset column, its data type, and its business meaning.

### 📄 Project_Report.pdf *(Coming Soon)*
A comprehensive report summarizing the project methodology, analysis, key findings, machine learning results, and business recommendations.

## Purpose

The documentation helps users understand the project beyond the source code by explaining:

- Project workflow
- SQL logic
- Dataset structure
- Business insights
- Analytical approach
- Final recommendations
