# 📂 Dataset

This folder contains the raw and processed datasets used in the project.

## Files

- **drivers.csv** — Historical Formula 1 driver statistics used for SQL analysis and Python-based machine learning.

## Dataset Features

The dataset includes information such as:

- Driver Name
- Nationality
- Seasons
- Championships
- Race Entries
- Race Starts
- Pole Positions
- Race Wins
- Podiums
- Fastest Laps
- Points
- Active Status
- Championship Years
- Decade
- Performance Rates
- Points per Entry
- Champion Label

## Purpose

The dataset serves as the foundation for:

- SQL Data Analysis
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Predictive Modeling
