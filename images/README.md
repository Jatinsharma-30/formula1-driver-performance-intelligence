# 🖼️ Images

This folder contains screenshots and visualizations generated during the **Formula 1 Driver Performance Intelligence** project. These images provide visual evidence of the analyses, machine learning evaluation, and key findings presented throughout the project.

---

# 📁 Contents

| Image | Description |
|-------|-------------|
| **Champion Distribution.jpeg** | Pie chart showing the distribution of champion and non-champion drivers. |
| **Confusion Matrix.jpeg** | Confusion matrix illustrating the classification performance of the selected machine learning model. |
| **Correlation Matrix.jpeg** | Correlation matrix displaying relationships between numerical variables in the dataset. |
| **Feature Importance.jpeg** | Feature importance chart highlighting the most influential variables for championship prediction. |
| **ROC Curve.jpeg** | Receiver Operating Characteristic (ROC) curve used to evaluate model performance. |
| **Top 10 Drivers by Race Wins.jpeg** | Bar chart showing the ten most successful drivers based on total race victories. |
| **Top 10 Nationalities by no. of Drivers.jpeg** | Bar chart showing the countries that have produced the highest number of Formula 1 drivers. |

# 📊 Purpose

These visualizations support the analytical findings presented throughout the project by:
