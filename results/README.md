# 📊 Results

This folder contains the outputs generated throughout the **Formula 1 Driver Performance Intelligence** project. These results summarize the findings from SQL analysis, Python exploratory data analysis, feature engineering, and machine learning.

The files in this directory provide reproducible outputs that support the project's conclusions and can be used for reporting, visualization, and further analysis.

---

# 📁 Folder Contents

| File | Description |
|------|-------------|
| **engineered_dataset.csv** | Dataset after feature engineering and preprocessing, ready for machine learning. |
| **model_metrics.csv** | Performance metrics of all machine learning models evaluated during the project. |
| **feature_importance.csv** | Relative importance of each feature in the final machine learning model. |
| **model_evaluation.csv** | Final evaluation metrics of the selected machine learning model. |
| **summary_statistics.csv** *(Optional)* | Summary statistics generated during exploratory data analysis. |
| **predictions.csv** *(Optional)* | Sample predictions produced by the trained machine learning model. |

## Feature Importance

The trained Decision Tree model identifies the variables that contribute most to championship prediction.

Typical high-impact features include:

- Race Wins
- Podiums
- Points
- Pole Positions
- Years Active

Feature importance helps explain how the model makes predictions and improves interpretability.
